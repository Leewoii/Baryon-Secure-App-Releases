# Baryon-Secure-App-Releases

This repository is only for public release distribution files for **Baryon Secure**.

It contains:

- signed updater metadata (`latest.json`)
- signed update artifacts (`.sig` and platform bundles/installers)

It does **not** contain:

- application source code
- private configuration
- internal build scripts
- secrets

Source code is maintained in a separate private repository.
